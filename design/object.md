The MSK Item Formula (The Mold)

1. Header Information

Name: The unique title of the item.

Theme / Archetype: The associated power source (e.g., Artificer, Oceanic, YHWH).

Uniqueness: Is this a Unique Item? (Only one can exist in the game universe).

Attunement Time (MoT): The "Age" value added to the deck's total for power brackets.

2. Classification & Handing

Object Type:

Handheld: Must be held in a hand to function.

Wearable: Assigned to a specific slot (Head, Torso, or Hand/Arm).

Handing: Specifies if it requires one, two, or no hands (for certain wearables).

Usage Type:

Permanent: Reusable throughout the match.

Consumable: Has a set number of Uses before it is discarded.

3. Requirements & Balancing Logic

Physical Requirements: Does it need another item or technique to function? [User Query].

Deep Requirements: Archetype depth (number of cards in memory), HP thresholds, or specific Marks.

4. The "One Thing" Power

Passive Effect: A constant benefit while equipped/held (e.g., "Allows use of Telekinetic techniques").

Active Effect: A one-time or triggered effect (e.g., "Sharpen a weapon to deal +3 damage").

Sustain (Debt): If the item creates a lasting effect on the board, it may incur a flat stamina debt per turn until the player Breathes.

5. Scientific / Fantasy Logic

The "Why": The narrative or mechanical logic behind the item's function (e.g., "Liquid Oxygen propulsion" or "Magnetic refraction").
