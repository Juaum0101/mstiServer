1. Header Information (Identification)

Name: The unique title of the technique.

Theme / Archetype: The specific power source (e.g., Alchemist, Fearweaver, Sworkskills). This is used for Archetype Depth checks.

Age (MoT): The "Months of Training" value used for deck-building power brackets.

2. Resource Costs

Stamina Cost: The immediate energy required to activate the card. Variable Cost (X:X): Denoted if the player can perform a Stamina Influx to boost effects (e.g., 2 Stamina per +1 Damage).

Cost Type: Defines how the cost is paid: Once: A single payment upon activation.

Sustained: A flat amount removed from the stamina pool at the start of every turn while active.

Charge Time (Optional): The number of turns or phases the player must wait before the effect resolves.

3. Handing & Equipment (Physical Requirements)

Handing: Specifies if the technique is:

Unhanded: Requires no hands (e.g., Gaze or Breath-based).

One-Handed / Two-Handed: Requires specific hands to be free.

Weapon-Tied: Requires a specific weapon type (e.g., Dagger, Greatsword).

Item-Required: Requires a specific item in the Inventory or equipped (e.g., Ceramic Crucible).

4. Deep Requirements (The Balancing Logic)

This section is mandatory for "Stronger" techniques but often empty for "Basic" stepping stones.

Archetype Depth: Minimum number of cards of the same theme required in the Memory (Deck).

Health Threshold: Requires HP to be above or below a certain value (e.g., "Only works if HP < 10").

Mark / Mutation Check: Requires a specific King Mark or Mutation to be active on the character.

State Condition: Requires the player to be in a specific state (e.g., "Must be Hovering," "Must not have Dodged last turn").

5. Technical Execution (Action Data)

Action Type: (e.g., Attack, Defense, Movement, Projection, Abstract, Breathing).

Range (Units): The distance the technique covers (e.g., Melee, 3 Units, Sight).

Range Type: The geometry of the effect (e.g., Straight, Cone, Around, AOE).

6. Effects & Logic

Active Effect: The primary "one thing" the card does (e.g., "Applies 1 layer of Fear").

Passive Effect (Optional): An ongoing benefit while the card is held in Active Memory or while the effect is sustained.

Status Keyword (Optional): Any status it applies (e.g., Burning, Staggered, Blind).

7. Side Effects & Drawbacks (Optional Balancing)

Action Lockout: Prevents specific actions next turn (e.g., "Cannot Dodge next turn").

Reaction Vulnerability: Negative effects if the player is attacked while using the card (e.g., "If hit this turn, take +2 damage").

8. Flavor & Lore

Description: A brief narrative explanation of how the technique looks or its history.
