The MSK Weapon Formula (The Mold)

1. Header Information

Name: Unique title.

Category: (e.g., Dagger, One-Handed Sword, Two-Handed Sword, Shield, Bow, Orbs).

Attunement Time (MoT): The "Age" value added to the deck's total for power brackets.

2. Physical Handling & Properties

Handing: One-Handed, Two-Handed, or Any-Handed (specifying bonuses for 2-handing).

Weapon Rating (WR): The base value for damage and defense.

Weight/Stamina Impact (Optional): Specific bonuses or penalties to stamina consumption when used (e.g., "+1 Stamina to Dodge while equipped").

3. Combat Capabilities

Range:

Melee: Standard is 1 unit.

Ranged: X Units (specifying pathing: Straight, Diagonal, Boomerang, etc.).

Weapon Properties: Innate traits (e.g., "Ranged weapons: WR -1 when defending; WR -2 when melee attacking").

Ammunition (For Ranged): Specifies if it is ammunition-dependent and what it consumes (arrows, items, energy).

4. Innate Abilities

Passive Effects: Constant benefits while equipped (e.g., "Increases range of Touch actions").

Weapon Techniques: Cards that are added to your Active Memory only while this weapon is held.

5. Scientific/Fantasy Logic

The "Why": A brief note on how the weapon functions according to the world’s rules (e.g., magnetic return, pressurized oxygen).
